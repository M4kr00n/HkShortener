<h2 align="center">🗂 Instalação</h2>

```
💻 Termux

$ pkg upgrade && pkg update -y
$ pkg install git
$ pkg install ncurses-utils -y
$ git clone https://github.com/M4kr00n/HkShortener
$ cd HkShortener
$ chmod +x *
$ bash install-termux
$ ./HkShortener

💻 Ubuntu

$ apt upgrade && apt update -y
$ apt-get install curl git
$ git clone https://github.com/M4kr00n/HkShortener
$ cd HkShortener
$ chmod +x *
$ bash install-ubuntu
$ ./HkShortener

```
